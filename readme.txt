== Installation ==

### The Easy Way

1. Go to Plugins > Add New in your WordPress dashboard
2. Search for "Short URL"
3. Click "Install Now" and then "Activate" (two clicks to awesome!)

### The DIY Way

1. Download the ZIP file from our releases page
2. Go to Plugins > Add New > Upload Plugin
3. Choose the downloaded ZIP and click "Install Now"
4. Activate and you're ready to roll!

== Frequently Asked Questions ==

= Can I use my own domain for short links? =

Absolutely! You can use your main domain or set up a custom domain just for short links.

= Is Short URL GDPR compliant? =

Yes! We offer IP anonymization and configurable data retention periods to help you stay compliant.

= Can I import links from another URL shortener? =

= Will Short URL slow down my site? =
